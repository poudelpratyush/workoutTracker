# Workout Tracker Mobile Application

A full stack workout tracking application desined for me to log, monitor and, track workouts and acheive personal goals.

## Tech Stack
- Frontend: React Native, Expo
- Backend: Java, Spring Boot
- Database: PostgreSQL

To view

- Node.js(v18+)
- Java 17+
- Expo Go app installed on your mobile phone or an emulator

## Backend Setup (Spring Boot)

### 1. Navigate to the server
```
cd backend
```
### 2. Configure your enviroment variables. Update src/main/resources/application.properties

### 3. Start the Spring Boot server
```
./mvnw spring-boot:run
```
## Frontend Setup (React Native/Expo)

### 1. Open a new terminal and navigate to the frontend directory
```
cd frontend
```
### 2. Install the dependencies
```
npm install
```
### 3. Update the API base URL to local machine so that backend can connect with frontend.

### 4. Start the Expo development server
```
npm start
```
